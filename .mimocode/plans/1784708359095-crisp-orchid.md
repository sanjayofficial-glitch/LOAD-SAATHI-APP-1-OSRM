# LoadSaathi SEO Fix Plan

## Summary

Fix critical SEO issues: wrong domain in OG tags, missing sitemap, incomplete robots.txt, no structured data. **3 files touched, zero auth/logic changes.**

---

## Changes

### 1. `index.html` — Replace `<head>` meta block (lines 3–27)

Replace everything from `<meta charset="UTF-8" />` through `<title>...</title>` with the comprehensive SEO block. Keeps the error handler script, favicon, manifest, viewport, and body untouched.

**Key fixes:**
- `og:url` / `og:image` / `twitter:image`: `loadsaathi.app` → `loadsaathi.in`
- New: `<meta name="keywords">`, `<meta name="robots">`, `<link rel="canonical">`
- New: JSON-LD structured data (Organization + WebSite schemas)
- Updated: title, description, og:title, og:description, twitter:* tags

### 2. `public/robots.txt` — Replace entirely

```
User-agent: *
Allow: /

Disallow: /dashboard
Disallow: /profile
Disallow: /settings
Disallow: /admin
Disallow: /api/

Sitemap: https://loadsaathi.in/sitemap.xml
```

Replaces the existing per-bot rules with a clean blanket rule + disallow list + sitemap reference.
