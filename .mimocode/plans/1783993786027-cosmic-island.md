# Plan: Integrate Real Fuel Price API for Accurate Cost Estimation

## Current State

The app has a basic emissions calculator (`src/utils/emissions.ts`) that uses hardcoded CO₂ emission factors (0.32/0.68/0.92 kg/km for light/medium/heavy vehicles). It shows CO₂ estimates on trip/shipment detail pages via `EmissionsCard`. There is **no fuel cost calculation** — only emissions.

The user wants to integrate **real diesel prices** from Indian cities to enable accurate fuel cost estimation for trips.

## Fuel Price APIs Available

### Option A: GitHub Open API (Free, no key)
- **Base URL**: `https://fuelprice-api-india.herokuapp.com/`
- **Endpoints**:
  - `GET /states` — list all states
  - `GET /<state>/districts` — list districts in a state
  - `GET /price/<state>` — all fuel prices in a state
  - `GET /price/<state>/<district>` — price for specific district
- **Returns**: Petrol + Diesel prices per district with price change info
- **Caveat**: Hosted on Heroku free tier — may be slow/unreliable

### Option B: indianapi.in (Requires API key, paid sandbox)
- More reliable but requires API key from indianapi.in
- User mentioned they will provide the API key

**Decision**: Use GitHub open API as primary (free, no key needed). Build indianapi.in integration as a fallback path — user can add their API key later via `VITE_INDIANAPI_KEY` env variable. For now, only the GitHub API is active.

```typescript
// Functions:
fetchFuelPrice(state: string, district: string): Promise<FuelPriceResult>
```

### Step 2: Create Fuel Cost Calculator Utility

**File**: `src/utils/fuelCost.ts`

```typescript
// Calculates estimated fuel cost for a trip
// Uses: distance, vehicle type → fuel efficiency (km/L), diesel price

interface FuelCostInput {
  distanceKm: number;
  vehicleType?: string | null;
  originState?: string;
  originDistrict?: string;
  loadTonnes?: number;
}

interface FuelCostResult {
  dieselPricePerLitre: number;
  fuelEfficiencyKmpl: number;  // km per litre
  estimatedLitres: number;
  estimatedFuelCost: number;   // ₹
  loadedEfficiencyKmpl: number; // efficiency with load
  source: 'api' | 'fallback';
}

// Fuel efficiency defaults by vehicle class (loaded):
// light (Tata Ace/Tempo): 8-10 km/L
// medium (14-17 ft): 4-6 km/L
// heavy (17+ ft): 2.5-4 km/L
```

## Key Design Decisions

1. **Caching**: Cache fuel prices in localStorage for 6 hours (API has rate limits)
2. **Fallback**: If API fails, use ₹90/L as national average diesel price
3. **No auth needed**: The GitHub API is public, no key required
4. **Lazy loading**: Fetch fuel price only when EmissionsCard is visible (not on page load)
5. **Vehicle efficiency**: Use industry-standard km/L values by vehicle class, adjustable by load weight
