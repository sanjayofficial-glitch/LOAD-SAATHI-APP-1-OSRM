### Part 4: Blog Content Structure for AI Extractability (Priority: Medium)

Current blog articles are plain `<p>` tags — no headings, no internal links, no structured blocks. AI systems extract passages better with clear structure.

**4a. Add H2/H3 headings to blog articles**
- File: `src/pages/blog/BlogArticle.tsx`
- Change content format from `string[]` to support headings + paragraphs
- Add a `type` field: `{ type: 'paragraph' | 'heading', text: string }`
- Render H2/H3 for heading elements

**4b. Add internal linking between blog articles**
- Add `relatedArticles` field to each article
- Render a "Related Articles" section at the bottom
- Cross-link between articles on similar topics

**4c. Add "Key Takeaways" box to each article**
- Self-contained summary block (40-60 words) optimized for snippet extraction
- Rendered as a highlighted box at the top of each article

**File to modify:** `src/pages/blog/BlogArticle.tsx`
